# 做菜小程序

一个专注于分享美食制作方法的微信小程序，与后台管理系统配套使用。

## 功能特性

### 🏠 首页
- Banner轮播展示
- 分类导航
- 热门菜品推荐
- 为你推荐菜品

### 📱 核心功能
- **菜品浏览**：按分类浏览菜品，支持分页加载
- **菜品详情**：查看详细制作步骤、食材清单、营养信息
- **制作指导**：分步骤制作指导，支持图片预览
- **搜索功能**：支持菜品名称、食材搜索，保存搜索历史
- **收藏功能**：收藏喜欢的菜品，本地存储
- **个人中心**：用户信息管理、统计数据展示

### 🎯 特色功能
- **智能搜索**：实时搜索建议，热门关键词推荐
- **制作步骤**：详细的图文制作步骤，支持全屏查看
- **食材清单**：完整的食材用量列表
- **难度标识**：简单、中等、困难三级难度标识
- **数据统计**：浏览量、收藏量等数据展示

## 技术架构

### 前端技术
- **框架**：微信小程序原生开发
- **样式**：WXSS + Flex布局
- **数据管理**：本地存储 + 网络请求
- **组件化**：模块化页面设计

### 后端对接
- **API接口**：RESTful API设计
- **数据格式**：JSON数据交换
- **图片处理**：MinIO文件服务
- **分页加载**：支持大数据量分页展示

## 页面功能

### 首页 (index)
- Banner轮播图展示
- 分类快速导航（最多8个）
- 热门菜品横向滚动展示
- 推荐菜品网格展示
- 下拉刷新数据

### 分类页面 (category)
- 分类列表展示
- 分类菜品列表
- 上拉加载更多
- 热门菜品专区

### 菜品详情页 (dish-detail)
- 菜品基本信息展示
- 制作步骤详细说明
- 食材清单完整列表
- 制作步骤弹窗指导
- 收藏/分享功能

### 搜索页面 (search)
- 实时搜索功能
- 搜索历史记录
- 热门搜索推荐
- 搜索结果分页展示

### 收藏页面 (favorites)
- 收藏菜品列表
- 取消收藏功能
- 清空所有收藏
- 空状态引导

### 个人中心 (profile)
- 用户信息展示
- 数据统计（收藏、浏览、搜索）
- 功能菜单
- 缓存管理

## 数据存储

### 本地存储
- `userInfo`: 用户信息
- `favorites`: 收藏菜品列表
- `searchHistory`: 搜索历史
- `viewHistory`: 浏览历史

### 接口数据
- 菜品数据从后台管理系统获取
- 支持实时数据更新
- 图片通过MinIO服务加载

## 开发说明

### 环境要求
- 微信开发者工具
- Node.js (用于后端服务)
- 后台管理系统运行

### 配置说明
1. 修改 `app.js` 中的 `baseUrl` 为实际后端地址
2. 在微信开发者工具中导入项目
3. 配置合法域名（开发时可跳过）
4. 启动后台管理系统服务

### 接口对接
小程序通过以下接口与后台系统对接：
- `/banner/list` - 获取Banner列表
- `/category/list` - 获取分类列表
- `/dish/page` - 获取菜品分页数据
- `/dish/{id}` - 获取菜品详情
- `/dish-step/list/{dishId}` - 获取制作步骤
- `/dish-ingredient/list/{dishId}` - 获取食材列表
- `/dish/search` - 搜索菜品
- `/file/preview` - 图片预览
